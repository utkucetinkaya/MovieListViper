# MovieListViper

## About ##
My application of popular series list that I show to the screen through Alafomire and Kingfisher libraries using TMDB API. I built the application with VIPER architecture.
- Swift
- UIKit
- TableView
- Alamofire
- Kingfisher
- VIPER
